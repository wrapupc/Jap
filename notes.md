形容词/形容动词
```
１.形容词可以作为谓语(名词要借助だ)
空は青い
空は青いです


2.形容词+名词不需要の
青い空

3.形容动词作谓语需要だ
教室は静かだ
教室は静かです

4.形容动词+名词需要な
静かな教室

5.连用形(连接用言(形容词,形容动词,动词 助动词,小句))
5.1形容词用く　(い=>く)
教室は大きく、静かだ。
5.2形容动词用で
教室は静かで、大きい。

6.否定(い=>く)
この本は面白くない。(面白い因为要连接ない, 需要用连用形面白く)
この本は面白くないです。
この本は面白くありません。

7.过去时
7.1.1形容词过去时(かった)
かったです

7.1.2过去否定
なかった
なかったです
ありませんでした(でした是だった礼貌形式,表示过去时)

7.2名词/形容动词过去时(だった)
私はもともとがくせい学生だった。
私はもともとがくせい学生でした(でした表示礼貌)。

昨日教室は静かではなかった。(ではない=>ではなかった)
昨日教室は静かではありませんでした。

8.连体词(只能连接体言)
大きな
小さな
色んな
同じ(也可以是形容动词)

```

动词
```
1.自动词(intangible)/他动词(tangible)
1.1.规则:他动词->自动词(的倾向)
    す->eる->aる
    eる->五段动词

2.~が(提示主语) ~を(提示宾语)
私が走る

私がご飯を食べる

3.五段动词(汉字+一个假名几乎都是)
あ=>未然形
い=>连用形
う=>终止形/连体形
え=>假定形/命令形
お=>未然形

4.一段动词(去掉一个假名后剩下一个い/え段的音的几乎都是)
去掉う=>未然形/连用形
る=>终止形/连体形
え=>假定形
お=>命令形

5.活用(カ变/サ变动词)
5.1.カ变:
く=>未然形
き=>连用形
くる=>终止形/连体形
くれ=>假定形
こい=>命令形

5.2.サ变:
し、せ、さ=>未然形
し=>连用形
する=>终止形/连体形
すれ=>假定形
しろ、せよ=>命令形
```
