形容词/形容动词
```
１.形容词可以作为谓语(名词要借助だ)
空は青い
空は青いです


2.形容词+名词不需要の
青い空

3.形容动词作谓语需要だ
教室は静かだ
教室は静かです

4.形容动词+名词需要な
静かな教室

5.连用形(连接用言(形容词,形容动词,动词 助动词,小句))
5.1形容词用く (い=>く)
教室は大きく、静かだ。
5.2形容动词用で
教室は静かで、大きい。

6.否定(い=>く)
この本は面白くない。(面白い因为要连接ない, 需要用连用形面白く)
この本は面白くないです。
この本は面白くありません。

7.过去时
7.1.1形容词过去时(かった)
かったです

7.1.2过去否定
なかった
なかったです
ありませんでした(でした是だった礼貌形式,表示过去时)

7.2名词/形容动词过去时(だった)
私はもともとがくせい学生だった。
私はもともとがくせい学生でした(でした表示礼貌)。

昨日教室は静かではなかった。(ではない=>ではなかった)
昨日教室は静かではありませんでした。

8.连体词(只能连接体言)
大きな
小さな
色んな
同じ(也可以是形容动词)

```

动词
```
1.自动词(intangible)/他动词(tangible)
1.1.规则:他动词->自动词(的倾向)
    す->eる->aる
    eる->五段动词

2.~が(提示主语) ~を(提示宾语)
私が走る

私がご飯を食べる

3.五段动词(汉字+一个假名几乎都是)
あ=>未然形
い=>连用形
う=>终止形/连体形
え=>假定形/命令形
お=>未然形

4.一段动词(去掉一个假名后剩下一个い/え段的音的几乎都是)
去掉う=>未然形/连用形
る=>终止形/连体形
え=>假定形
お=>命令形

5.活用(カ变/サ变动词)
5.1.カ变:
く=>未然形
き=>连用形
くる=>终止形/连体形
くれ=>假定形
こい=>命令形

5.2.サ变:
し、せ、さ=>未然形
し=>连用形
する=>终止形/连体形
すれ=>假定形
しろ、せよ=>命令形
```

动词未然形
```
======あ段未然形======
1.~ない
1.1.动词后面加ない时,要用未然形
五段动词:
行く => 行かない
待つ => 待たない

一段动词:
見る => 見ない
起きる => 起きない
食べる => 食べない
カ/サ变动词:
来る => 来ない
する => しない

2.语态(可能态,被动态,使役态)都要用未然形
2.1.~(ら)れる(可能态)
2.1.1.五段动词要用あ段未然形+れる(现代日语这里会被简写为eる)
行く => 行かれる => 行ける
待つ => 待たれる => 待てる

2.1.2.一段动词未然形+られる
見る => 見られる
起きる => 起きられる
食べる => 食べられる

2.1.3.カ变动词未然形+られる
来る => 来られる
する => できる(现代日语特殊情况) 






2.2.~(ら)れる(被动态)
2.2.1.五段动词要用あ段未然形+れる
行く => 行かれる
待つ => 待たれる

2.2.2.一段动词未然形+られる
見る => 見られる
起きる => 起きられる
食べる => 食べられる

2.2.3.カ变动词未然形+られる
来る => 来られる

2.2.4.サ变动词未然形+れる
する => される






2.3.~(さ)せる(使役态)
2.3.1.五段动词あ段未然形+せる
行く => 行かせる
待つ => またえる

2.3.2.一段动词未然形+させる
見る => 見させる
起きる => 起きさえる
食べる => 食べさえる

2.3.3.カ变动词未然形+させる
来る => 来させる
する => さえる

2.3.4.サ变动词未然形+せる



======お段未然形======
1.~(よ)う(意志形)
1.1.五段动词お段未然形+う
行く => 行こう
待つ => 待とう

1.2.一段动词未然形+よう
見る => 見よう
起きる => 起きよう
食べる => 食べよう

1.2.カ/サ变动词未然形+よう
来る => 来よう
する => しよう
```
